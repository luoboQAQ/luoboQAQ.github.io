在Linux下安装SVP4

记录在Arch下安装SVP4的过程以及踩的坑

2022-08-23
阅读时长: 2 分钟

之前在《如何优雅的看番》这篇文章中我提到了补帧的3款软件，在Windows系统上我使用的是dmitriRender这款软件，而其并没有提供Linux版本的程序。与之相反的是，SVP不但提供了Linux版本，而且还是免费的。

由于官方文档是英文且有些地方没有表述清楚，这里我就将我安装和配置的方式记录下来供大家参考。

安装

  1. 对于Arch用户非常简单，直接yay -S svp-bin即可完成安装。

    对于其他发行版，可以选择在官网上下载并安装。

  2. 安装播放器，这里我推荐安装VLC，因为其安装简便，并且后面配置SVP也很容易。

    安装直接输入sudo pacman -S vlc即可

  3. 如果需要GPU加速，还需要安装相应的插件sudo pacman -S ocl-icd。我之前就是没有安装，导致一直无法读取到我的显卡。

    安装好后就可以在SVP菜单——应用程序设置——GPU加速里看到你的显卡了

配置

  1. 修改VLC插件目录权限，以便SVP写入插件。

    1

    sudo chmod 777 /usr/lib/vlc/plugins/video_filter

  2. 在SVP的菜单里选择“工具”——“VLC的SVP插件”，将其选中

  3. 检查/usr/lib/vlc/plugins/video_filter目录下的libsvpcontrol_plugin.so和libvapoursynth_plugin.so是否有可执行权限，我之前就是因为没有给执行权限，导致VLC一直不会调用SVP

    1
2

    chmod +x /usr/lib/vlc/plugins/video_filter/libsvpcontrol_plugin.so
chmod +x /usr/lib/vlc/plugins/video_filter/libvapoursynth_plugin.so

  4. 官网表示需要开启“反交错”才会启用，但实测好像不开启也可以运行。

    如果VLC偏好设置——显示设置——全部——控制界面有SVP control interface

    以及滤镜里有Vapoursynth filter即表明安装成功了。

小结

官网上的安装文档写了很长，但实际上对于VLC使用SVP并不复杂，复杂的是MPV使用SVP。本来打算配几张图的，但实际写下来也没有发现需要图片辅助说明的，那就算了，相信聪明的你一定能看懂 _(:з)∠)_
